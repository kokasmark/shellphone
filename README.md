A Terminal based **Terraria** Character Editor.

### Feautures
- Player base stats
- Inventory
- Armor/Dyes/Accessories
- Banks
- Buffs

### Planned Feautures
- Command based inventory editing
  - `give Item x999 to inventory`
- Hot reloading character by calling LoadPlayer from Terraria.exe
